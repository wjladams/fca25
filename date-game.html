The Date Game

Using the digits of today's date — 9/22/2025 — to build a true equation.

How it works

  1. Write the date as digits in order: 9, 2, 2, 2, 0, 2, 5.
  2. Place operations between the digits: +, −, ×, ÷, exponent (^), and parentheses.
  3. You may insert exactly one equals sign = somewhere in the sequence.
  4. Do not reorder the digits. They must appear left‑to‑right as 9, 2, 2, 2, 0, 2, 5.
  5. Concatenation is allowed: you may join adjacent digits to form a multi‑digit number as long as you keep the order (e.g., 2 and 5 may become 25).
  6. Use every digit exactly once. The equals sign splits the expression into two sides that must be equal.
Tip: Start by trying to place the equals sign so that both sides have similar totals when using only addition and subtraction. Then experiment by replacing a pair like 2 + 2 with 2 × 2 or 2^2 to fine‑tune the balance.

Classroom ideas

  • Have students find any valid equation, then challenge them to find one using a new operation.
  • Compare strategies: Where did they place the equals sign? Which operations helped balance both sides?
